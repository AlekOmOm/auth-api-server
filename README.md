# Simple Auth Server

## usage

```Bash
cp .env.template .env
# add values to .env
```

```Bash
make setup-frontend && make dev-frontend # bash

# in another terminal run:
make setup-backend && make dev-backend # bash
```

powershell 
```powershell
make setup-frontend; make dev-frontend

# in another terminal run:
make setup-backend; make dev-backend
```
